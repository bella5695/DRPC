Recent Changes

Here are the recent changes made to this blog by DRPC Intern Isabelle Yam

These are the recent changes that I've made to the website for usability, presentation, engagement, and design of the website.

Menu

The menu has changed to have different buttons. When you click on the menu, it shows which page you are on. The home page is a logo of the DRPC when you click on the logo, it directs you to the home page on the menu. This is to seperate the different pages, and highlight the page that you are on. It also is to make the home page simple and direct the user to the homepage when you click on the logo. There is a sub menu for certain parts of the menu with a drop-down menu style. These are to access the pages within the main category of type of pages.

Home Page

The home page has altered to description of the company on it, with a picture of the logo beside it to describe the homepage in a simple manner.

In the Field

This page has changed to show the awards, grants and presentations from the DRPC alongside an image that represents their activity in the field, alongside a description of what they did. This is to see the awards, grants, and presentations next to what they did in a unique and visual way.

Teaching Resources

This page has altered to show the teaching resources on the page through an image that represents the topic, and a descprtion on the right, where on the left it provides a preview PDF of that teaching resource. These changes were to display the teaching resources in a simple manner and provide a description of each one.

Reading Lists

This features two coloumns, with an image of the reading list, description of the reading lits, provided by the list of readings in each reading list. Also, there is a download button to download the summary of each reading list. Then followed by a leave a review of a reading form at the bottom. There are links to certain readings on a website.These changes were to display the reading lists, and show what each one is about, and the ability to download it. Also, to leave a review of each reading.

Using Resources

This page is designed to showcase the examples of how to cite with the DRPC using preformatted text design in the example, and an image about using digital resources to inspire the audience. This link was to show how to cite with the DRPC in a way that was easy to understand and engaging for the reader.

Privacy Week

The Privacy Week page has altered to animations, when you scroll the events will pop up, with the image of the poster, next to the description of the event, with time, date, and location. At the end there is a table that summarizes the events where you can click on that event in the table to bring you back to where to sign up for the event on the page. This is to showcase the events that are available in a eye-catching way to the audience with the poster to show the image, and the events. The table is so that the user can register for the one that they want at the end of seeing all the events in a summarized table.

Get Involved

The Get Involved page is showing the ways to get involved with the DRPC in the form of text beside a picture with the different methods.

Blog

This page is changed to show the recent blog posts on the right, and a description of the recent blog posts on the right. Then there is a table of monthly calendar where you can preview the blog posts for that month. This was to show the recent blog posts, as a blog page for the reader to be able to read, and on the right be able to download the recent blog post. Also, the table was to be able to view all the blog posts in each month.

Newsletter

This newsletter page shows interaction with the ability for the user to be able to have the option to choose the newsletter by table, pdf, or month by calendar. There is a few pdf previews to show the recent pdfs. This was to show the different ways to access the newsletters through a table to download, and show the recent pdf previews to show the newsletters, as well as,the option to view the recent newsletters on the side. There is also a search bar to search the newsletter by keywords available as an option to search for the newsletter.

Digital Resources

This page shows the digital resources by the DRPC, on one side, and lists the digital resources on the left side. The digital resources are displayed through link, image, and youtube video. This page is to show the digital resources of the DRPC in a user-friendly way.

Contacts

The contacts page that I did, included the description for contacting the DRPC, followed by social media links and a contact form.
